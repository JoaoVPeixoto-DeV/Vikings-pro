# Vikings-pro
Sistema de consultoria esportiva em desenvolvimento utilizando Python.
Vikings Pro

O Vikings é um sistema em Python voltado para acompanhamento de treinos, nutrição e evolução física.

Funcionalidades

- Cálculo de IMC
- Definição de déficit e superávit calórico
- Organização de treinos personalizados
- Módulo de hipertrofia
- futuro módulo de powerlifter
- futuro módulo de definicão
- Sistema de menu interativo
- Estrutura modular em Python

Objetivo do projeto

O objetivo do Vikings Pro é funcionar como um assistente pessoal de treino e evolução física, ajudando o usuário a acompanhar desempenho e rotina de forma simples e direta, rumo ao objetivo escolhido

Tecnologias utilizadas

- Python
- Bash (scripts auxiliares)
- Estrutura modular com múltiplos arquivos

Estrutura do projeto

- vikingspro.py → arquivo principal
- parte1.py → lógica de cálculo e dados
- menu_principal.py → interface de menu
- hipertrofia.py → módulo de treino
